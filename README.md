### その他のプロジェクト
- **[かんじもん](https://kanji.jr-genius.jp)** - 小学1年生の漢字80字をキャラクター化した学習アプリ
- **[あそびラボ](https://asobi-lab.jr-genius.jp)** - 遊びと学びを融合させた実験的コンテンツサイト
